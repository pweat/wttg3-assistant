# Site screenshots

Structure:

```
assets/sites/<site-slug>/<page-slug>/1.webp
assets/sites/<site-slug>/<page-slug>/2.webp
...
assets/sites/<site-slug>/_meta.json
```

- Screenshots are WebP (`1.webp`, `2.webp`, …)
- `_meta.json` holds display name, page list, and notes (site + per-page)
- Doughy has notes only (no screenshots)
- Sources: Steam guide by MazKnight + local uploads
